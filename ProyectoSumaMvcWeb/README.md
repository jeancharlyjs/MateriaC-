# ProyectoSumaMvcWeb

Proyecto ASP.NET Core MVC en C# para Visual Studio.

## Contenido

- `SumaMvcWeb`: aplicacion MVC Web que solicita dos numeros y muestra la suma.
- `SumaMvcWeb.Tests`: proyecto MSTest con 3 pruebas unitarias.
- `ProyectoSumaMvcWeb.sln`: solucion principal para abrir en Visual Studio.

## Ejecutar la aplicacion

```bash
dotnet run --project SumaMvcWeb/SumaMvcWeb.csproj
```

## Ejecutar las pruebas

```bash
dotnet test SumaMvcWeb.Tests/SumaMvcWeb.Tests.csproj
```

En Visual Studio tambien puedes usar:

```text
Test > Test Explorer > Run All Tests
```
